# Omniedge-android

### Prerequisites
1. Install NodeJs (suggest version 12+)
2. Install Amplify CLI by running:
```shell script
npm install -g @aws-amplify/cli
```

### Build project
1. clone repo
2. git checkout dev_android
3. git repo fold
4. git submodule init
5. git submodule update
6. ./gradlew :app:assembleDebug
